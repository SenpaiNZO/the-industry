Installation

Etape 1

Une fois votre achat effectué et vérifié, vous téléchargez maintenant votre application simplement comme tout téléchargement banal.

Etape 2

Votre application devrait se trouver dans le dossier des téléchargements et porterait le nom predictor-aviator.apk

Une fois votre application repérée, vous pouvez passez à l'étape suivante.

Etape 3

Maintenant, il faut accepter l'installation d'applications de sources inconnues dans les paramètres de votre téléphone et vous pouvez enfin installer le Predictor Aviator
